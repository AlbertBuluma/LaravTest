# LaravTest

A sample API project built using the Laravel framework.


### Get started

``` 
$ git clone
$ cp .env.example .env
$ composer install
$ php artisan migrate
$ php artisan db:seed
$ php artisan serve
```

### Run tests:

``` 
$ ./vendor/bin/phpunit
```
